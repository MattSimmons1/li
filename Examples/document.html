# Welcome to _lightweight_ Word formatting with zero effort! ### To View 1. Save with _.html_ extension, e.g. _'my notes.html'_ 2. Open in any web browser with JavaScript enabled ### To Edit 1. Open in any raw text editor (Notepad on Windows, TextEdit on Mac) 2. Enter text beneath the script tag on the first line 3. Use [Markdown](https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet) to format the text or add images
